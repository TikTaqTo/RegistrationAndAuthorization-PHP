# RegistrationAndAuthorization-PHP
Php registration and authorization system
<br>Начальные настройки: 
<br>1)Перейдите в папку vendor, найдите файл db.php добавьте название вашей бд и пароль.
<br>2)В папке vendor найдите файл signup.php и используйте свои сервис отправки сообщений на почту, я использовал mailtrap.
